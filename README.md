# Raksha AI - Shield Against Deepfakes & Digital Arrest

An AI-powered real-time security application designed to protect users from live video deepfakes, cloned voice scams, and psychological digital arrest frauds.

## Tech Stack (Planned MVP)
- **Frontend:** HTML5, Tailwind CSS, JavaScript
- **Backend:** Python (FastAPI / Flask)
- **AI Models:** HuggingFace Deepfake Detection API, OpenAI Whisper (for voice analysis)
- **Database:** Local Storage / Firebase
